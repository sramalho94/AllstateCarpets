# AllstateCarpets

## Getting Started

- `clone` down repo
- `cd` into project repo
- run `npm i`
- run `npm run dev`

## Getting Started with Docker

- run `docker pull sramalho94/allstate-carpets:0.0.7.PRODUCTION` using Docker CLI
- start the image

## Technologies used

- next.js
- tailwind.css
- node.js
- docker
